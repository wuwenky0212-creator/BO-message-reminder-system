# RCS 5.0 消息提醒系统

一个基于 Vue 3 + FastAPI 的金融系统消息提醒平台。

🚀 **在线演示**：https://bo-message-reminder-system-h2wuyih8.vercel.app

## 🌟 功能特性

- ✅ 实时消息提醒（6种异常类型）
- ✅ 多页面业务流程管理
- ✅ 债券持仓风险预警
- ✅ 响应式设计，专业金融 UI
- ✅ WebSocket 实时推送（计划中）

## 🚀 快速开始

### 本地开发

**前端**
```bash
cd frontend
npm install
npm run dev
```
访问：http://localhost:5173

**后端**
```bash
cd backend
python demo_server.py
```
访问：http://localhost:8000

### 在线部署

查看 [DEPLOYMENT.md](./DEPLOYMENT.md) 获取详细的 Vercel + Render 部署指南。
